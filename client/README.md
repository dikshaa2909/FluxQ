### ● Dashboard Overview
FluxQ Dashboard is a high-performance clinical monitoring system built with React 19. It provides real-time visualization of AI-optimized patient triage, simulation of doctor assignments, and in-depth performance analytics.

### ● Key Features

#### ┃ 0x01 ― Smart Triage Dashboard
› Clinical intake workflow for specialized physical queues (Diabetes, BP, Cholesterol, etc.).
› High-precision symtom assessment with real-time feedback.

#### ┃ 0x02 ― Live Simulation Engine
› Real-time doctor assignment simulation with burnout and throughput tracking.
› Dynamic wait-time estimation based on current clinical load.

#### ┃ 0x03 ― Visual Performance Analytics
› Interactive comparison charts (Optimized vs. FCFS metrics).
› Fairness reports and starvation risk alerts.

#### ┃ 0x04 ― Premium UX Framework
› Developed with a high-end glassmorphism design language.
› Real-time connectivity status via the **Live Dot** system.
